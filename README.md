## 프로젝트 설명
- `header`와 `footer`는 화면이 렌더링될 때 자동으로 로드됩니다.
- `header`의 검색 기능과 메뉴 하단의 이동 선 구현이 필요합니다.
- `footer`는 아직 구현되지 않았습니다.
---------
